Steps to run the code using TurboC++ software:

1.Install TurboC++
2.Go to Projects folder inside TURBOC3 folder where it is installed(Default path : C:\TURBOC3\Projects)
3.Paste the source files in the projects folder.
4.Open TurboC++
5.Click on open source file
6.Select the source file you want to run
7.Press ctrl+f9 to execute.



Steps to run the code online :

1.Open the sourcefile with text editor.
2.Copy the code
3.Open the website "https://www.tutorialspoint.com/compile_c_online.php"
4.Paste the code and remove getch() at the last but one line.
5.compile.
6.execute.
